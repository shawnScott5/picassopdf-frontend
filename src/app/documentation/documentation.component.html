Authentication

All API requests require authentication using your API key. Include your API key in the Authorization header of your requests.

Using your API key 
curl -X POST https://api.picassopdf.com/v1/convert \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{{ '{' }}
    "url": "https://example.com",
    "format": "pdf"
  {{ '}' }}'

‚ö†Ô∏è Keep your API key secure

Never expose your API key in client-side code or public repositories. Always use environment variables or secure configuration management.

PDF Conversion

Convert web pages, HTML content, or documents into high-quality PDFs with custom styling and options.

Convert a Web Page

Basic web page conversion 
const response = await fetch('https://api.picassopdf.com/v1/convert', {{ '{' }}
  method: 'POST',
  headers: {{ '{' }}
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  {{ '}' }},
  body: JSON.stringify({{ '{' }}
    url: 'https://example.com',
    format: 'pdf',
    options: {{ '{' }}
      pageSize: 'A4',
      margin: '20mm',
      printBackground: true
    {{ '}' }}
  {{ '}' }})
{{ '}' }});

const result = await response.json();
console.log(result);

Convert HTML Content

Convert HTML string to PDF 
const response = await fetch('https://api.picassopdf.com/v1/convert', {{ '{' }}
  method: 'POST',
  headers: {{ '{' }}
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  {{ '}' }},
  body: JSON.stringify({{ '{' }}
    html: '<h1>Hello World</h1><p>This is a test document.</p>',
    format: 'pdf',
    options: {{ '{' }}
      pageSize: 'Letter',
      margin: '15mm',
      header: '<div style="text-align: center; color: #666;">My Document</div>',
      footer: '<div style="text-align: center; color: #666;">Page {{ '{' }}page{{ '}' }} of {{ '{' }}total{{ '}' }}</div>'
    {{ '}' }}
  {{ '}' }})
{{ '}' }});

Conversion Options

Parameter Type Default Description
url string - URL of the webpage to convert
html string - HTML content to convert
format string pdf Output format (pdf)
pageSize string A4 Page size (A4, Letter, Legal, etc.)
margin string 10mm Page margins
printBackground boolean true Include background graphics
header string - Custom header HTML
footer string - Custom footer HTML

Rate Limit Headers

Every API response includes rate limit information in the headers:

X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1640995200

‚ö†Ô∏è Rate Limit Exceeded

When you exceed your rate limit, the API returns a 429 Too Many Requests status code. Wait until the reset time before making additional requests.

Error Handling

PicassoPDF uses conventional HTTP response codes to indicate the success or failure of API requests.

HTTP Status Codes

Code Status Description
200 OK Request successful
201 Created Resource created successfully
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
403 Forbidden Insufficient permissions
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error

Webhooks

Receive real-time notifications when your conversions are completed or encounter errors.

Setting Up Webhooks

Configure webhook endpoint 
const response = await fetch('https://api.picassopdf.com/v1/webhooks', {{ '{' }}
  method: 'POST',
  headers: {{ '{' }}
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  {{ '}' }},
  body: JSON.stringify({{ '{' }}
    url: 'https://your-domain.com/webhook',
    events: ['conversion.completed', 'conversion.failed'],
    secret: 'your-webhook-secret'
  {{ '}' }})
{{ '}' }});

Webhook Events

Event Description
conversion.started Conversion process has begun
conversion.completed Conversion completed successfully
conversion.failed Conversion failed with error
